# История обновлений FineFrameController
## v.1.3
* Исправлены ошибки масштабирования
* Добавлено центрирования окна видео относительно интерфейса
## v.1.2
* Скорректирован расчет размера окна приложения, чтобы интерфейс не перекрывал окно просмотра
* Добавлено масштабирование видео до размеров интерфейса для возможности комфортной работы с видео низкого разрешение кадра 
## v.1.1
* Кнопка сохранить теперь сразу переходит к следующему кадру, чтобы уменьшить количество нажатий клавиш при обработке видео
* Добавлена поддержка кириллицы при сохранении кадров путем замены метода сохранения изображений с openCV на Pillow 
* Уменьшен объем сохраняемых кадров без потери качества
## v.1.0
Первая версия программы представленная общественности
