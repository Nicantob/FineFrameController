# FineFrameController

Программа представляет собой удобный инструмент для работы с видеофайлами формата MP4 и AVI, предназначенный для анализа и редактирования видеозаписей путем пошагового перемещения по кадрам. Приложение выполнено на языке программирования Python с использованием популярных библиотек Tkinter (для построения графического интерфейса) и OpenCV (для обработки видеоданных).

## Функционал программы:

- **Отображение видеофайлов**: Поддержка форматов MP4 и AVI. После загрузки видео первым делом открывается предварительный кадр.
- **Просмотр и прокрутка кадров**: Быстрая навигация по отдельным кадрам видео благодаря стрелкам “←” и “→” или “a” и “d”, либо ручное задание номера кадра в специальном поле.
- **Управление воспроизведением**: Простая система управления воспроизведением видео, включающая паузу, продолжение воспроизведения и стоп.
- **Покадровая навигация**: Помимо стандартного воспроизведения, предусмотрена удобная возможность покадрового перемещения вперёд и назад.
- **Редактирование и экспорт кадров**: Есть возможность сохранять отдельные кадры в виде PNG-файлов. Имя файла формируется автоматически на основе названия оригинального видео и порядкового номера кадра.
- **Настройки воспроизведения**: Возможность настройки интервала времени между кадрами (задержки) вручную через соответствующее поле.
- **Удобный интерфейс**: Простые иконки и понятные надписи делают использование программы удобным и быстрым.
- **Использование горячих клавиш**: Программа поддерживает удобные сочетания клавиш для быстрой навигации и выполнения команд (Ctrl-o — открыть видео, пробел — воспроизведение/пауза, s — стоп и переход к заданному номеру кадра, w — сохранение кадра, стрелки или a и o — навигация по кадрам).

## Интерфейс программы:

Интерфейс организован таким образом, чтобы обеспечить быстрый доступ ко всем основным действиям.  Под основным окном для отображение видео расположены управляющие кнопки, включая открытие файла, воспроизведение, паузу, сохранение кадра и покадровое перемещение, а также поля для выбора номера кадра и задержкой между воспроизведением кадров.

## Типичные сценарии использования:

- Анализ видеофайлов по частям для выявления деталей.
- Покадровый разбор записей спортивных соревнований, выступлений артистов, съёмочных материалов и других мероприятий.
- Редактирование видеофайлов с возможностью точного выделения конкретных моментов.

Программа отлично подходит для тех, кому важно точное позиционирование и контроль над каждым элементом видеоряда.
